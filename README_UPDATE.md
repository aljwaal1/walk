# تحديث تطبيق رفيق المشي - Android v1

هذه الحزمة مبنية على تعليقك على تطبيق `walk`:

- تحسين سرعة ظهور التطبيق عند الفتح.
- تصغير الدائرة الكبيرة في الصفحة الرئيسية.
- إلغاء دوران الدائرة السريع عند البداية، وجعلها مؤشر تقدم ثابت.
- جعل شاشة المشي الرئيسية أهدأ وأقصر، بحيث تظهر المعلومات الأساسية في نفس الصفحة قدر الإمكان.
- إضافة أيقونة Android خاصة للتطبيق.
- تعديل GitHub Actions لتوليد أيقونات Android قبل بناء APK.

## طريقة الاستخدام

1. فك الضغط داخل جذر مستودع `walk`.
2. تأكد أن الملف التالي موجود: `assets/icon/app_icon.png`.
3. شغّل:

```bash
python tools/apply_walk_update_v1.py
```

4. ارفع التعديلات إلى GitHub.
5. شغّل Workflow باسم `Build APK Direct`.

## الملفات المهمة

- `assets/icon/app_icon.png`: أيقونة التطبيق.
- `tools/apply_walk_update_v1.py`: سكربت يطبق التعديلات على `lib/main.dart` و `pubspec.yaml` و workflow.
- `replacement_walkview_compact.dart.txt`: نسخة شاشة المشي الجديدة للرجوع اليدوي إذا احتجت.

## ملاحظة مهمة

لم أرسل APK جاهزًا لأن بيئة الملفات هنا لا تستطيع الوصول مباشرة إلى GitHub لبناء المشروع كاملًا. هذه حزمة تعديل جاهزة للرفع على المستودع وتشغيل GitHub Actions.
