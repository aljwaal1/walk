# مساعد المشي والحركة — رفيق المشي v2

تطبيق Flutter عربي يحفز المستخدم على المشي اليومي، بتصميم فاتح ورياضي وألوان حيوية، مع تتبع يعمل حتى مع قفل الشاشة.

## الجديد في v2

- **تصميم جديد بالكامل**: خلفية فاتحة، ألوان حيوية (أخضر/تيل/أزرق/برتقالي)، بطاقات وظلال أنيقة.
- **التتبع في الخلفية**: يستمر تتبع GPS والمؤقت حتى عند قفل الشاشة، عبر خدمة أمامية (Foreground Service) مع إشعار صامت بسيط أثناء الجلسة فقط (مطلوب من نظام أندرويد).
- استرجاع الجلسة تلقائياً إذا أُغلق التطبيق أثناء التتبع.
- شريط دائري للتقدم بدل الخطي، مع عرض السرعة والمسافة بشكل أوضح.
- رسم بياني أسبوعي محسّن.
- شارات إضافية للتحفيز (25 جلسة، 14 يوماً متتالياً).
- صفحة تواصل مع المطور مع نسخ احتياطي تلقائي للرسالة إن لم يفتح تطبيق البريد.

## المميزات الأساسية

- تحدي مشي باختيار الوقت (5 إلى 45 دقيقة).
- عداد وقت دائري مباشر.
- حساب المسافة والسرعة عبر GPS.
- تسجيل الجلسات محلياً (بدون إنترنت).
- حساب الأيام المتتالية (Streak).
- 8 شارات تحفيزية.
- سجل أسبوعي ورسوم بيانية.

## طريقة البناء عبر GitHub

ارفع الملفات إلى GitHub، ثم شغّل workflow باسم:

`Build APK Direct`

بعد نجاح البناء سيظهر ملف APK مباشرة داخل:

`apk/walking-companion.apk`

وسيظهر ملف `DOWNLOAD_APK.md` وفيه رابط التحميل المباشر.

## ملاحظات مهمة

- التطبيق Android فقط.
- يحتاج صلاحية الموقع (يفضل "السماح دائماً" لتفعيل التتبع بالخلفية).
- عند بدء أول جلسة سيُطلب منك صلاحيتان: صلاحية الموقع، وصلاحية الإشعارات (لإظهار إشعار التتبع الصامت).
- الإشعار يظهر فقط أثناء الجلسة النشطة، بدون صوت أو رنين، ويختفي فوراً عند الإيقاف.
- الحد الأدنى المقترح: Android 6 وما فوق.
