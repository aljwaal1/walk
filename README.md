# داش كام بدون صوت

تطبيق Flutter Android لتسجيل الفيديو بدون صوت، مع طباعة التاريخ والوقت والسرعة فعلياً داخل ملف الفيديو المحفوظ.

## الميزات
- تسجيل فيديو فقط بدون ميكروفون (لا يطلب صلاحية الصوت إطلاقاً).
- اختيار الكاميرا الأمامية أو الخلفية.
- **ستامب محروق فعلياً داخل الفيديو** (وليس فقط على شاشة المعاينة): تاريخ ووقت متحرك ثانية بثانية + سرعة GPS حقيقية متحركة، بخط أحمر صغير أسفل الشاشة — يظهر في أي مشغل فيديو على أي جهاز.
- تقسيم التسجيل تلقائياً لمقاطع (المدة قابلة للتعديل من الإعدادات).
- حذف تلقائي للمقاطع القديمة عند تجاوز العدد المحدد، لتوفير مساحة التخزين.
- إشعار ثابت أثناء التسجيل، وإشعار عند حفظ كل مقطع.
- معرض لعرض وتشغيل ومشاركة وحذف المقاطع المحفوظة.
- تصميم داش كام احترافي (أسود/رمادي غامق + أحمر للتسجيل).

## كيف يعمل حرق الستامب
بعد كل تسجيل، يعالج التطبيق الفيديو عبر FFmpeg (مكتبة `ffmpeg_kit_flutter_new`) ويحرق طبقتين نصيتين:
1. التاريخ والوقت — يتحرك تلقائياً بالاعتماد على ساعة النظام لحظة معالجة كل فريم.
2. السرعة — تُحدَّث فعلياً كل ثانية بالقيم الحقيقية المسجَّلة من GPS أثناء القيادة، عبر ملف أوامر `sendcmd`.

هذه المعالجة تستغرق بضع ثوانٍ إضافية لكل دقيقة تسجيل وتظهر مؤشر "معالجة الستامب" في الشريط العلوي أثناء عملها.

## البناء
ارفع كل محتويات هذا المجلد (بما فيها `lib` و`.github`) إلى GitHub، ثم شغّل GitHub Actions.
سيظهر ملف APK باسم: `DashCam-No-Audio.apk`

ملاحظة: المحاولة الأولى للبناء قد تفشل تلقائياً وتُعاد مرة واحدة، لأن مكتبة FFmpeg تُنزّل ملفاتها الإضافية (AAR) في أول تشغيل لـ Gradle. هذا طبيعي ومتوقع، والبناء الثاني ينجح.
