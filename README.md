# مساعد المشي والحركة

تطبيق Flutter عربي خفيف يحفز المستخدم على المشي اليومي بدون تعقيد.

## النسخة الأولى

- تحدي مشي يومي: 10 أو 20 أو 30 دقيقة.
- زر بدء المشي وإيقاف الجلسة.
- عداد وقت مباشر.
- حساب المسافة المقطوعة عبر GPS.
- حساب متوسط السرعة.
- تسجيل الجلسات محليًا.
- حساب عدد الأيام المتتالية.
- شارات: أول مشي، 3 أيام، 7 أيام، 30 يوم.
- سجل أسبوعي بسيط.
- رسوم بيانية أعمدة لآخر 7 أيام: مدة المشي والمسافة.
- رسائل تشجيع.
- صفحة مراسلة المطور عبر الإيميل.

## طريقة البناء عبر GitHub

ارفع الملفات إلى GitHub، ثم شغل workflow باسم:

`Build APK Direct`

بعد نجاح البناء سيظهر ملف APK مباشرة داخل:

`apk/walking-companion.apk`

وسيظهر ملف:

`DOWNLOAD_APK.md`

وفيه رابط التحميل المباشر.

## ملاحظات

- التطبيق Android فقط.
- يحتاج صلاحية الموقع لحساب المسافة والسرعة.
- يعمل بدون إنترنت.
- تتبع GPS يعمل أثناء فتح التطبيق. التتبع في الخلفية يمكن إضافته لاحقًا.
- الحد الأدنى المقترح: Android 6 وما فوق.
